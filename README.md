# Telegram-бот для расчёта БЖУ, ХЕ и дозы инсулина

Это версия с:
- кнопками **Завтрак / Обед / Ужин / Перекус**;
- встроенной базой продуктов;
- расчётом **углеводов, белков, жиров, ХЕ** по каждому продукту и по всему приёму пищи;
- расчётом дозы по формуле через **суточный инсулин**;
- фото-режимом: бот пытается распознать еду по фото и оценить граммовку.

## Важно
Фото-режим — только приблизительная оценка. Перед уколом итог нужно проверить вручную.

## Что нужно
- Python 3.10+
- Telegram bot token
- для фото-режима: `OPENAI_API_KEY`

## Установка

### Windows PowerShell
Открой папку с ботом, затем **Shift + правая кнопка мыши** → **Открыть PowerShell здесь**.

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
$env:BOT_TOKEN="ТВОЙ_TELEGRAM_BOT_TOKEN"
$env:OPENAI_API_KEY="ТВОЙ_OPENAI_API_KEY"
python bot.py
```

Если фото не нужно, строку `OPENAI_API_KEY` можно не задавать.

### Windows CMD
```cmd
python -m venv .venv
.venv\Scripts\activate.bat
pip install -r requirements.txt
set BOT_TOKEN=ТВОЙ_TELEGRAM_BOT_TOKEN
set OPENAI_API_KEY=ТВОЙ_OPENAI_API_KEY
python bot.py
```

### Linux / macOS
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
export BOT_TOKEN="ТВОЙ_TELEGRAM_BOT_TOKEN"
export OPENAI_API_KEY="ТВОЙ_OPENAI_API_KEY"
python bot.py
```

## Как пользоваться
1. Нажми `/start`
2. Выбери **Завтрак / Обед / Ужин / Перекус**
3. Добавляй продукты:
   - из быстрых кнопок,
   - через категории,
   - поиском по названию,
   - или фото еды.
4. Нажми **✅ Рассчитать**
5. Введи сахар перед едой
6. Введи суточный инсулин за сегодня
7. Бот покажет:
   - Б/Ж/У,
   - ХЕ,
   - сколько ед. на 1 ХЕ,
   - точную дозу,
   - округлённую дозу под шаг шприца.

## Настройки
Есть кнопки:
- шаг шприца 1 / 0.5 / 0.1 ед;
- 1 ХЕ = 12 г или 10 г.

## Как включить в Telegram
1. Открой `@BotFather`
2. Создай бота через `/newbot`
3. Получи токен
4. Запусти `bot.py` с переменной `BOT_TOKEN`
5. Открой бота в Telegram и нажми `Start`

## Фото-режим
Фото-режим требует ключ OpenAI API. Бот отправляет фото в модель, которая поддерживает входные изображения, и получает список предполагаемых продуктов и граммовок. Затем эти продукты сопоставляются со встроенной базой и участвуют в расчёте.
